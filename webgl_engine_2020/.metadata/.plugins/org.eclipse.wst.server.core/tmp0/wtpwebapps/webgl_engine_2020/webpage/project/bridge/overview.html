拆装式桁梁

　

杆件列表    装配列表

 

拆装式桁梁是一种适用于铁路桥梁快速抢修的临时性可拼装、可拆卸的上承式桁梁。

　

一、基本结构

　

基本桁式为带立杆的华伦桁式（三角形桁架结构），桁高2.5米，节间2米，主桁中心距2米，跨度可按基本节间2米一进及按0.72米畸零跨度变换，主要弦杆、腹杆由槽钢、板材、角钢等组成U形截面，支杆、斜杆和立杆由角钢等组成L形截面，横联的上支杆采用由槽钢切焊而成的加强型，以增强桁架上弦杆的纵向稳定性。每片主桁为单腹式，上下弦截面由两个槽形杆件组成工字形截面，主桁间设有上、下平纵联及横向联结系。

　

二、适应范围

　

主要用于铁路中、小跨度桥梁战时抢修、平时防洪抢险的一种拆装桁梁，也可作为桥墩、桥台、导梁、施工脚手架、起重龙门架和其它临时结构之用。

　

三、技术条件

　

1. 设计荷载：按前进型单机随挂70KN/m或解放单机。不限速（特别注明者除外）。

2. 线路条件：单线、直线、曲线（特别注明者除外）。

3. 桁梁跨度：图号专桥0109（钢材材质为3号桥梁钢时）跨度16～32 m。专桥0113跨度为40～64 m，专桥0119（钢材材质为15MnTi时）跨度12～80 m。以上跨度均以2米一进，可按0.72 m变化畸零跨度。专桥0153包括拖拉架设导梁和32～64米拼装图。

4. 杆件重量：最大部件501号弦杆单重449.7千克，最小的33号栓钉单重0.46千克。用简单吊装工具即可搬运、拼装和拆卸。

　

四、桁梁特性

　

1. 互换性：全部杆件在工厂用机器样板生产，精确度高，同类杆件能完全互换，可多次装拆重复使用。用低合金钢制造的桁梁杆件、螺栓及栓钉共59种，用3号桥梁钢制造的桁梁杆件及螺栓、栓钉共44种。32米标准式桁梁杆件、螺栓及栓钉共36种。

2. 快速性：器材运达现场后，准备工作就绪，配备一般吊装起重工具，组织适当的作业人员能够在较短的时间内完成桁梁拼装和架设任务。

3. 机动性：拆装式桁梁最大部件450千克，用简单的吊装工具即可搬运，多数部件可用人力装、卸、拼、拆，装运方法机动灵活，一般情况下用汽车装运比较方便，不需要特殊措施。

4. 成套性：一般以32米或64米为一套成套生产和储备，全部杆件均由型钢或板材组成，规格品种符合国家标准。

　

五、拼装架设

　

桁梁的拼装，全部杆件在装运至工地后，首先整理宽度为五至六米、长度适当的作业场地，然后进行抄平，并用木枕或方木作为临时支承平台，在平台上预先画出主桁中心线以控制平直。

拼装方法根据人力、地形、起重工具而定，一般经短期培训的工人均能掌握快速拼装、架设的要求。桁梁拼装方法一般分为整体拼装（单向拼装、双向拼装）、分段拼装和分片拼装等多种方式方法。

1. 整体拼装，利用小型起吊机具如汽车吊、简易龙门吊在水平的支承平台上先组拼下弦杆、下平联，再立杆、斜杆、横联杆，然后利用吊车拼装上弦杆及上平联。

2. 分片拼装，在平整的场地上先成片拼装，然后利用滑轮钢丝绳或人字扒杆将两片主桁起立对合，再安装横联、下平联和上平联。

3. 分段拼装，先将主桁分段预拼成约8m的一段，然后再组装成孔。

 

架梁入桥位，一般梁前端可利用原杆件组拼导梁，下弦在辕轴上拖拉入桥位，再落梁定位。

　

六、使用保养

　

拆装式桁梁作为一种战备交通运输保障抢修器材，专业保障队伍应定期培训演练,掌握使用性能，拼装、拆卸杆件、螺栓和栓钉时，不可重锤打击，以防杆件变形和造成螺纹损坏及栓杆弯曲，使用后应及时清理污泥、锈渍、上油，损坏的及时补配。
