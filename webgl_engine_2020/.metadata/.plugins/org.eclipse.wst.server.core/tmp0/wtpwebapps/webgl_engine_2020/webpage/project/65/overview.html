六五式铁路军用桥墩是一种主要用于抢修中等跨度、中高高度桥梁的拆装式成套制式器材，也可在新建铁路工程中用作便桥桥墩，架设临时支墩和膺架、脚手等，还可组成简易起重设备和用于公路桥梁抢修。若节点联结改用高强度螺栓和铆合，可作为半永久或永久性建筑物使用。

全套基本器材共有杆件、配件19种(CI～Cig)，紧固件3种。同一编号的杆件、配件可以互换使用，部分不同编号的杆件、配件，还可以顶替使用。

墩身高度可以按0.5米模数调整，适应性强。除基本器材外，还配有小型吊装设备，使拆装作业灵便、迅速。杆作种类少，重量轻，最大单元重量仅477．4公斤，适于铁路、公路运输，也可以人力短途抬运。由于具有结构简单，互换性强，拆装迅速、运送方便等优点，己在抢修和施工生产中得到广泛使用。

　

　

一、设计规范和规定

　

l.  铁路桥涵设计规范(铁道部，一九五九年本，以下简称“桥规”)

2. 铁路破损桥涵临时及短期修复设计技术规程(一九五一年)。

　

二、设计范围

　

1. 以适应战时铁路临时性桥墩抢修为主，并考虑平时铁路桥梁灾害的应急修复和新建工程中的多种用途。

2. 设计墩身高度为10一40米；也可以用于40米以上高度，但应另行检算。

3. 设计梁跨为16一32米六四式铁路军用梁和拆装式桥梁；用于大跨度梁时；应另行检算。

4. 线路条件，按标准轨距、单线考虑；必要时，也可用于双线或窄轨。

5. 基础按木桩基础、卧木基础、混凝土基础考虑。必要时，也可在残存墩身上通过适当连接后拼组六五式铁路军用桥墩。基础应按实际情况具体设计。

　

三、设计荷载

　

按主力(恒载、竖向活载、冲击力、离心力)及附加力(制动力、风压力、列车摇摆力、水流冲击力)的最不利组合进行计算；本设计未考虑地震力、冰压力和船舶撞击力，遇有上述情况时，应采取适当防护措施和另作检算。

1. 主力

(1)恒载；恒载分结构物自重和静水压力(即浮力)两类，当计算桥墩的稳定性时，应考虑静水压力。

(2)竖向活载：中华人民共和国铁道标准活载中一20级。计算桥墩横向稳定性时。空车荷载按1吨／米均布活载计算。

(3)冲击力：只在计算上垫梁及立柱的承载力时才考虑冲击力的影响。

(4)离心力：桥在曲线上的离心力按作用在轨顶以上2米处的横向均匀分布载重计算。

2. 附加力

(1)制动力：按竖向静活载重量的10％计算。其作用点可自轨顶上2米处移到支座中心处而不计算所产生的竖向附加反力。当计算桥墩时：

①对于六四式铁路军用梁，其制动力按平均分配于梁的两端支座上考虑。

②对于拆装式桥梁，按墩顶较大梁跨处设固定支座，其制动力全部作用于该固定支座上计算。

(2)风力

①横向风力：混凝土基础，不论桥上有车或无车，横向风力均按0.125吨／平方米计算。木桩基础或卧木基础，桥上有车时横向风力为 0.080吨／平方米；桥上无车时横向凤力为 0.100吨／平方米。

②纵向风力，受风面积及风力作用点均按“桥规” 有关规定办理。

(3)列车横向摇摆力：按“‘桥规”有关规定办理。

(4)水流冲击力：按“桥规”对矩形实体桥墩的规定计算。

　

四、水文地质条件

　

1. 计算水深：混凝土基础按无水和水深10米两种情况，木桩基础按水深8米和4米两种情况，卧木基础只考虑无水情况进行计算。

2. 计算流速：3米/秒。
