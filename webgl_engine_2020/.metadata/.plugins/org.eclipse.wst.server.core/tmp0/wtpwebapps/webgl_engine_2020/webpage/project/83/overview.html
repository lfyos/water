八三式铁路轻型军用桥墩（简称八三式桥墩）是根据1977年6月国务院、中央军委常规装备发展领导小组[1977]24号文，自行研制的一种桥墩抢修成套制式器材。1983年9月原铁道兵指挥部召开技术鉴定会议通过，并经原铁道兵军工产品定型委员会批准设计定型。

八三式桥墩全套器材杆件种类少，单元重量轻使用率高，并且拆装迅速，储运方便。

　

1976年12月．原铁道兵科研所向铁道兵司令部报送了“铁路轻型军用桥墩课题论证报告”。1977年3月，铁道兵司令部召开桥梁抢修座谈会，审议通过了这个报告。同年6月，国务院、中央军委常规装备发展领导小组批准进行研制，并下达了主要战术技术要求。原铁道兵和国家建委确定铁道兵科研所和陕西金属结构厂为负责研制单位。1978年1月，铁道兵司令部开方案设计审查会，审查批准了研制方案。同年7月，铁道兵科研所完成试制技术设计。1980年12月，陕西金属结构厂完成试制。1981年5月，铁道兵司令部召开了产品出厂评议会，批准试制产品合格出厂。1979年完成“多柱式塔架结构模型试验”专题理论研究项目。1981年9月，由原铁道兵舟桥团在山东齐河完成拼装速度试验，检验了战术性能要求。1982年9月，在津浦线(天津――浦口)新建明光大桥工地，完成了现场静动载通车试验，检验了技术性能要求。1983年9月，铁道兵军工产品定型委员会批准初步设计定型。

　

一、基本用途

　

主要适用于战时、平时中小跨度、中低高度桥梁，特别是铁路便桥墩的应急、快速抢修，也可在新建铁路工程中用作便桥桥墩、临时支墩和鹰架脚手等，也可用于公路桥梁的快速抢修和组成简易起重设备。

　

二、技术性能

　

　

1. 适应性

　

(1)线路条件：单线、直线、曲线半径≤500m、坡度<6‰。

(2)梁跨配合：10～32m的工字钢梁，六四式铁路军用梁和拆装式桁梁。

(3)设计墩高：5～15m，可以按0.5m变化遁身高度。

　

2. 快速性：准备工作就绪，器材运到现场，使用配套的工具，39人人力拼装，可以在6小时内完成一座10m高度的桥墩结构的安装任务。

　

3. 机动性：主要单元为线性杆件，最大不见重253kg，长3.5m，完全可以用人力装、卸、拆、拼。全部器材可用普通卡车装运，且具有较大装载系数，便于运输、储存。

　

4. 简便性：零部件种类仅9种，不同杆件易于识别，适合夜间作业。器材轻便、结构简单。

　

5. 互换性：同类零部件能完全互换，可多次拆、装、运和重复使用。

　

6. 成套性：可成套研制，成套生产，成套储备，成套使用。

　

7. 地基基础：适应卧木、混凝土和木桩基础使用，必要时能用于残墩接高。

　

8. 生产适应性：材料品种规格便于供应，符合国家标准化要求。适于一般金属加工厂战时动员生产。。

　

三、桥墩拼装

一般可利用配套的吊装设备、安装工具和一定数量的通用吊装机具安装在两根立柱上组成两组起吊扒杆同时作业互相提升，从下到上完成全部安装工作。
为了加快拼装速度，在地形条件允许的情况下，对于墩身高度低于10米的桥墩杆件部分，也要采用一次倒顶法进行拼装。
拼装前首先对基础做必要的处理，力求平整，不平处应使用薄铁片将垫梁垫平，保证立柱的垂直度。
