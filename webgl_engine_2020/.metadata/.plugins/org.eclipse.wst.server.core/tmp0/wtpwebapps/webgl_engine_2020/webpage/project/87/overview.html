八七型铁路应急抢修钢梁

八七型铁路应急抢修钢梁（以下简称八七型梁）1976年至1982年由铁道兵研究试制，1985年至1986年拼装及静动载实验完成的一种新型上、下承式、可快速装、拆轻便钢梁，它具有跨度大、桁式多样、适应性广等特点，是一种桥梁抢修制式器材。

　

一、适应范围

主要用于战时铁路桥梁抢修和平时意外情况的应急抢修，还可以在铁路、公路等基建工程中作桥梁、预架梁以及正线上的短期使用。

　

二、主要技术条件

1.       适应性：铁路单线、直线、简支、多跨连续，基本跨度40~64米，扩大使用到96米，跨度变化范围1米，可以组成连续梁。

2.       设计活载：前进型机车单机随挂70kn/m，均布荷载，跨度40~80米时列车运行速度为30~40km/h，跨度80米以上时，列车运行速度为15km/h。

3.       净空界限：桥限――2甲。

4.       拼装架设速度：准备工作就绪，器材运到现场，可以在72小时内拼组架设一孔64米跨度梁。

5.       基本构件种类少，单元最大重量轻，器材储存体积小，适宜于铁路车辆和普通卡车的装运，同类构件、连接件完全互换；不同构件，联结易于识别，适应多次拆、装、运和重复使用。

6.       可成套生产，成套储备，成套使用。除结构本体外，备有配备的拼装专用机具。

7.       器材品种规格全部国产化，标准化，便于供应，适于一般钢梁工厂，金属结构厂等战时动员生产。

　

三、基本结构和组成

1.       主梁为桁式结构，主桁基本型为简支梁，标准节间长度4米，另可利用基本部件组成1米及两米的畸零节间以调整跨度或组成再分桁架对主桁进行加固。上、下承梁均设以纵横梁体系，纵梁中心距18米。桥面用I甲普枕平放，支座采用聚四氟乙烯盒式橡胶组合支座。

2.       基本部件36种，单元最大长度7.98米，单元最大重量1746kg。梁的联结采用剪力承压型螺栓，光杆直径30毫米，孔径31毫米。

　

四、现场架设、施工方法

拼组设计系按在桥头路基进行，并充分利用预拼组以缩短拼装时间，上承式梁的拼组采用由本器材部件组成的龙门桁车或通用吊车，穿式梁的宽度、高度较大，宜用较大型设备吊装。

架设设计，以纵向拖拉作为通用架设方法；同时，考虑浮运、浮拖、悬臂拼装等多种方式。多孔梁架设时可以连续拖拉，上承式梁用落梁龙门架吊落，落架时间短，也比较安全，下承式梁宜用千斤顶落梁，拖拉用专用拖拉滚轮组，拼装工具及施工辅助部件随同基本部件配发。
